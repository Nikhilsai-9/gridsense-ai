# GridSense AI Demo Script

## Short Human-Style Description

GridSense AI is a simple smart meter intelligence system for BESCOM. It helps find unusual power usage, estimate revenue loss, and support officer action in a clear and easy way.

## Simple Video Script

Hello, this is our project GridSense AI.

This project is made for BESCOM.

The main idea is simple.

We use smart meter data to find unusual power usage.

If a meter suddenly behaves in a strange way, the system gives an alert.

It also shows the possible revenue loss and suggests what the officer can do next.

In this demo, we can see the meter data, the risk level, and the suggested action.

This is useful because it can help reduce power theft, save time, and support faster decisions.

It does not replace officers.

It helps them work in a smarter and easier way.
